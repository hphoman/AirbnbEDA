# Data Instructions

As mentioned in the project README, Inside Airbnb requests
that users download data directly from its website. To reproduce the
project, download the appropriate city files and place them within this repository. 

The csv files can be downloaded [here](https://insideairbnb.com/get-the-data/). Once on the website, 
scroll down to the bottom city of interest and select the "Show" button referenced in "Show archived
data". At the time of this project's creation, all five cities will have a csv file for September 2025,
and we recommend using this version. 

After downloading, please place all five csv files within this directory. You may need to either rename
the files in the expected syntax of code (that is, Austin.csv, Chicago.csv, LosAngeles.csv, NewYork.csv
, and Portland.csv) or change the `read_csv` parameter to the name of your files.
